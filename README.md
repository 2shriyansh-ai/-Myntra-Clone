# Myntra Clone Project
A simple Myntra‑style e‑commerce frontend built with HTML, CSS, and JavaScript.\
This project is a clone of Myntra’s product listing and product‑detail pages.  
It demonstrates basic HTML structure, CSS layout, and JavaScript interactivity for a shopping experience.

## How to view

1. Clone this repository:
   ```bash
   git clone https://github.com/2shriyansh-ai/myntra-clone-project.git
   
2. Open the folder in VS Code or any editor.

3. Open the main HTML file (for example index.html) in your browser.
